#  Spotify Clone (Music Player)

##  Description
This is a Spotify-inspired music player built using pure HTML, CSS, and JavaScript.  
It allows users to play songs, switch between tracks, control volume, and browse albums dynamically.

##  Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## Features
-  Play / Pause music
-  Next / Previous song control
-  Dynamic song loading from folders
-  Album-based song browsing
-  Song duration & progress bar (seekbar)
-  Volume control + Mute option
-  Responsive UI with sidebar toggle

##  How It Works
- Uses JavaScript `Audio()` object to control music playback
- Fetch API is used to dynamically load songs and albums
- DOM manipulation is used to render song lists and UI updates
- Custom seekbar implemented using event listeners
